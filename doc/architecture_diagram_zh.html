vision_grasp / 架构总览

ROS 2 视觉抓取流水线

面向 Piper 机械臂的单包流水线：仿真或真实相机采集、基于 HSV 颜色分割的目标检测、 像素到基座坐标变换、IK 驱动的轨迹规划（含抓取状态机）、以及可选的串口驱动执行。 仿真优先设计；所有参数集中在 config/params.yaml 中。

运行时概览

  • 单个 ROS 2 包：5 个可执行节点 + 2 个共享模块。
  • 仿真优先：相机渲染彩色色块，无需硬件。
  • 所有参数集中在 config/params.yaml。

数据流

  • /image_raw → /detected_objects → /grasp_target → /joint_states
  • 像素 (u,v) → 射线-平面求交 → IK 求解 → 关节插值轨迹
  • 抓取状态机: IDLE → OPEN → APPROACH → DESCEND → CLOSE → LIFT → RETRACT

夹爪约定

  • joint7 使用负值闭合。
  • joint8 使用正值闭合。
  • 网格关于 X=0 平面对称。
  • IK 运动链: base_link → link6（6 自由度）。
